Обхождане на дърво

Във файла drv_1.cpp се използва процедурата preorder, която реализира обхождане на дървото в дълбочина.

void preorder(link t) {

if (t) {

cout<<t->key<<“ “;

preorder(t->l);

preorder(t->r);

}

}

Останалите два начина за обхождане в дълбочина са

void inorder(link t) {

if (t) {

preorder(t->l);

cout<<t->key<<“ “;

preorder(t->r);

}

}

void postorder(link t) {

if (t) {

preorder(t->l);

preorder(t->r);

cout<<t->key<<“ “; }

}
