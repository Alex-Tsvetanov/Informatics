Run Your Business in Real-Time with SAP HANA
Is the data you're basing business decisions hours, days or weeks old? Imagine if you knew when, where, and how customers were buying without delay. View inventories, supply chains, or financials with up to the second accuracy. Spot market changes early and adjust with precision. This is running in "real-time". SGI helps make this possible with SAP HANA at near limitless scale and mission-critical reliability.

Achieve Greater Insight. Gain Competitive Advantage.
SGI UV 300H enables running SAP Business Suite, SAP Bank Analyzer, SAP S/4HANA Finance, and other business-critical applications on SAP HANA. It combines OLTP and OLAP workloads and eliminates the time-consuming extract, transform, and load processes to generate real-time reports on demand.

"Our implementation of the 12TB SGI UV 300H in our In-Memory Research Laboratory has freed researchers to experiment with enterprise applications on very large in-memory databases and predictive applications."
- Hasso Plattner, Prof. Dr. h.c. mult., Hasso Plattner Institute.

Run Simple. Run Big.
Perform complex joins, run multiple analytic engines simultaneously (including text, geo-spatial, and live data streaming), and eliminate silos. Plus with the SGI UV 300H there are no cluster nodes, cluster network, or SAN to configure and administer, and no need for data re-balancing.

Scale Up vs. Scale Out

Scale Up Seamlessly in a Single Node
Featuring Intel® Xeon® E7-8800 v3 and v4 processors and 7th generation SGI NUMAlink technology, SGI UV 300H is SAP-certified under general availability from 4 to 20 sockets with up to 20TBs of in-memory computing capacity as a single node - making SGI UV 300H #1 in scale-up capacity for SAP HANA. Pending certification, it's designed to scale even further with up to 32 sockets and 32TB of shared memory!

Depend on Mission-critical Reliability
Equipped with robust RAS features, SGI UV 300H delivers four 9's system resiliency. Single-node simplicity also extends to High Availability and Disaster Recovery with local automated system failover and remote system replication to achieve five 9's application availability.

Capitalize on Future-ready Design
The unique modular chassis architecture of the UV 300H enables you to start small and scale up in 4-socket increments. No over-provisioning. No forklift upgrades. Never outgrow.

Leverage SGI Solution Flexibility
Pre-installed with SUSE® Linux Enterprise Server or Red Hat Enterprise Linux for SAP HANA®*, SGI UV 300H can be deployed as an appliance pre-configured with SAP HANA** or in an SAP HANA Tailored Datacenter Integration (TDI) configuration. SGI UV 300H solutions are rapidly installed by an SGI systems engineer and supported 24x7 by SGI's renowned global services team.


* Currently E7 v3 processor only.

** SAP provides licensing.
